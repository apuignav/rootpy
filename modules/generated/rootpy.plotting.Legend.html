8.7. rootpy.plotting.Legend¶

class rootpy.plotting.Legend(nentries, pad=None, leftmargin=0.5, topmargin=0.05, rightmargin=0.05, entryheight=0.06)¶
AddEntry(thing, legendstyle=None, label=None)¶

Add an entry to the legend.

If legendstyle is None, it will be taken from thing’s ‘legendstyle’ attribute.

If label is None, the thing’s title will be used as the label.
